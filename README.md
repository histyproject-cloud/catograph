# Cartograph

세계관을 그리는 창작 허브 — 모바일 · 태블릿 · 데스크톱 완전 지원

## 반응형 레이아웃

| 화면 | 네비게이션 | 디테일 패널 |
|---|---|---|
| 모바일 (< 768px) | 바텀 탭바 | 바텀시트 (슬라이드업) |
| 태블릿 (768~1199px) | 햄버거 + 슬라이드 사이드바 | 오버레이 패널 |
| 데스크톱 (1200px+) | 고정 사이드바 | 우측 고정 패널 |

관계도 캔버스: 마우스 드래그 (PC) / 터치 드래그 + 핀치줌 (모바일·태블릿)

## 시작하기

### 1. Firebase 설정

`src/firebase.js`의 `YOUR_*` 값을 본인 Firebase config로 교체

### 2. Firestore 규칙

Firebase Console → Firestore → 규칙에 `firestore.rules` 내용 붙여넣기

### 3. 실행

```bash
npm install
npm start
```

## 기능

- Google 로그인
- 프로젝트 생성/삭제
- 캐릭터 추가/편집/태그
- 관계도 캔버스 (드래그 + 핀치줌 + 관계선)
- 복선 관리 (화수 추적 + 캐릭터 연결)
- 설정집 문서 에디터
